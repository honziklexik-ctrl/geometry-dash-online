# Geometry Dash — Made by me

This is a browser-based clone of the popular **Geometry Dash** game, made in HTML, CSS, and JavaScript.  
You can play it directly in your web browser — no installation required!

## Features

- Smooth player movement and jumping physics  
- Narrow, lethal spikes and moving obstacles  
- Collect coins for extra points  
- Multiple levels with different speeds and patterns  
- Particles, simple sound effects, and background music  
- Mobile-friendly and responsive canvas  
- High score tracking using localStorage  
- Single HTML file — easy to host anywhere

## How to Play

- Press **Space** or **Click/Tap** to jump  
- Avoid spikes and obstacles  
- Collect coins to increase your score  
- Try to beat your high score!

## Hosting

You can host this game using **GitHub Pages**, **Netlify**, or any static HTML hosting service.  
Simply upload the `index.html` file and share the link with your friends!
